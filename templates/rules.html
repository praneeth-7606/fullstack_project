File Format Requirements

Acceptable Formats

Recommended Formats:

  • ✓ .docx (Microsoft Word)
  • ✓ .pdf (Adobe PDF)

Avoid These Formats:

  • ✗ .doc (Legacy Word)
  • ✗ .rtf (Rich Text Format)
  • ✗ .txt (Plain Text)
  • ✗ Image files (.jpg, .png)

Resume Structure

Required Order of Sections

  1. Contact Information
  2. Professional Summary/Objective
  3. Work Experience
  4. Education
  5. Skills
  6. Additional Sections (Optional)

Pro Tip:

Keep your resume between 1-2 pages. For entry-level positions, stick to one page.

Required Sections Detail

Contact Information

Full Name Email Address Phone Number Location (City, State) LinkedIn Profile (Optional)

Format Requirements:

  • ✓ Use professional email address
  • ✓ Include country code for phone numbers
  • ✓ Write location as "City, State"

Work Experience

Company Name Job Title Dates (MM/YYYY - MM/YYYY) Location • Achievement/Responsibility 1 • Achievement/Responsibility 2

Important:

Use bullet points to list achievements. Start each bullet with an action verb.

Formatting Guidelines

Text Formatting

Recommended:

  • ✓ Font: Arial, Calibri, Times New Roman
  • ✓ Size: 10-12pt for body text
  • ✓ Headings: 14-16pt
  • ✓ Standard margins (1 inch)

Avoid:

  • ✗ Fancy fonts
  • ✗ Text boxes
  • ✗ Tables
  • ✗ Headers/Footers
  • ✗ Images/Graphics

Keywords & Skills

Keyword Optimization

How to Include Keywords:

Use relevant keywords from the job description naturally throughout your resume. Include both the spelled-out version and acronym where applicable (e.g., "Artificial Intelligence (AI)").

Skills Section Example:

Technical Skills: Python, Java, SQL, AWS Soft Skills: Project Management, Team Leadership Industry Knowledge: Agile Methodology, SDLC
